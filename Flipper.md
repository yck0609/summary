# Flipper

> Flipper 是一个用于在模拟器/模拟器或物理设备上调试 React Native 项目的平台工具。 它支持在 Android 和 iOS 上运行的项目，并可作为 macOS、Windows 和 Linux 上的桌面应用使用。
> 
> 它提供了各种功能，例如设备日志查看器、交互式原生布局检查器、网络检查器、本地数据库检查器、崩溃报告器等。 你可以在 Flipper 桌面应用中添加更多可用插件。
